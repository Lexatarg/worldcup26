# Security

This is a small, static personal project — a single `index.html` page hosted on GitHub Pages. There is no backend, no user accounts, no database, and no data collected from visitors. The page only displays publicly available match information.

## Reporting a problem

If you spot a security issue (for example, something in the page that could be misused), please open an issue in this repository, or use GitHub's private vulnerability reporting if it's something that shouldn't be public. I'll take a look when I can.

## Scope

Because the site is static and stores nothing about visitors, the realistic surface is small. Things worth flagging would include broken external links, anything that could mislead a visitor, or content that shouldn't be there.

## What this project does not do

- It does not collect, store, or share any personal data.
- It does not set tracking cookies.
- It does not require a login.
- It does not run any server-side code.
